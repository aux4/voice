#### Description

The `list` command displays all available text-to-speech voices installed on the system. An optional language filter narrows results by language code.

#### Usage

```bash
aux4 voice list [<language>]
```

--language  Filter by language code, e.g., en, pt, ja (default: show all)

#### Example

```bash
aux4 voice list
```

```bash
aux4 voice list en
```

```bash
aux4 voice list pt
```
